Lista de etiquetas de Sublime Text (Primera Parte)


Etiquetas de formato de texto
  1. Para colocar textos en Negrita debes escribir el texto < b > (Texto que deseas en Negrita) y < /b > al finalizar


  2. Si deseas escribir texto con cursiva debes escribir < i > (Texto que deseas en cursiva) y < / i >


  3. Para colocar textos subrayados debes escribir el texto < u > (Texto que deseas subrayado) y < /u >


  4. Si tienes que colocar saltos de linea se debe usar la etiqueta < br > al final de cada texto donde desees bajar o un salto de linea

  5. Al desear marcar un texto diferente puedes utilizar el < mark > para marcarlo de un fondo amarillo

  6. Buscando como tener un texto mas pequeño puedes utilizar el < small > para cear un texto de menor tamaño

  7. Para conseguir un texto tachado deberás usar < del > para conseguir un texto que luzca tachado

  8. Para conseguir tamaños de textos diferentes usarás < font size="Numero que desees" > y así podrás elegir el tamaño de texto de tu agrado

  9. Regresando con los textos más importantes puedes usar < strong > para conseguir un texto que se vea más importante que los otros

  10. Pero si buscas un texto con enfasis debes usar < em > y así conseguirás un texto con este estilo
