Contenidos Multimedia

En esta pagina aprenderás a colocar contenidos multimedia para tus paginas de HTML desde Sublime Text

El primer metodo para colocar contenidos multimedia que apenderás será el de las fotografias

  1. Al momento de querer colocar fotografias usarás el comando < P > < Img src=" " width="Número de anchura" height="Número de altura"> donde entre las comillas irá el enlace de la fotografia que deseas pegar desde google poniendo como ejemplo el siguiente texto < img src="https://imgflip.com/s/meme/Cute-Cat.jpg" width="200" height="400" > se generará la siguiente imagen

  2. Si deseas alinear la foto hacia el medio deberas colocar < P align="Center" al inicio del texto y al final colocar Align=2Middle > antes de cerra el signo >
    Por lo que el codigo se veria asi < P align="Center">< Img src="https://imgflip.com/s/meme/Cute-Cat.jpg" width="200" Height="400" Align=2Middle >


    y la imagen se mostraria de esta manera:


  3. Si deseas colocar una imagen con un link o enlace debes usar el codigo < a href="Enlace al que quieres enviar" > < img src="Imagen con la que se disfrazará el enlace". Donde seguido de esto irá de nuevo: width="Anchura que deseas en la imagen" height="Altura que deseas de la imagen">
    Poniendo un ejemplo seria: < a href="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRJ4rILCQrcBCGlHCybMXXkVDwJsq7kKF0-qpiz_Bnw28zEFQwb-5-xhkH4LtAgjMu8YEc&usqp=CAU"> < img src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRJ4rILCQrcBCGlHCybMXXkVDwJsq7kKF0-qpiz_Bnw28zEFQwb-5-xhkH4LtAgjMu8YEc&usqp=CAU" width="120" height="240" >
    Y la imagen mostrada seria la siguiente:
  4. Al buscar colocar un gif en una pagina web debes usar el codigo:
    < img src= "GifQueDeseas.gif " >
    Por lo que poniendo un ejemplo seria: < img src="https://i.gifer.com/47tv.gif" >
